# demo.docker

## step1：本地运行简单server
> 该工程是极简单的express node server，仅供演练；运行成功，通过浏览器访问会看到一句欢迎语。

1、按照依赖，根目录下执行：
```
npm install
```
2、如果本机安装cpm，执行（国内镜像，更快）：
```
cnpm install
```
3、访问：http://localhost:3000/  
你应该会看到: `"Hello, I'm a server response."`
